# Casino Royale

A free social-casino games showcase, built with Next.js, React, Three.js, and
Zustand.

## Games

- **Craps** — a full craps table with real physics dice (3D and 2D views), the
  complete betting layout (pass/come, odds, place, field, hardways, hop), and
  payouts you can actually see.
- **Crapless Craps** — the variant where you can't crap out on the come-out.

More games are on the way.

## Highlights

- Physics-driven 3D dice and table rendered with Three.js / React Three Fiber.
- An app-wide theme system — swap palettes across the lobby, felt, and 3D scene.
- A filterable game lobby with a shared casino-wide credit balance.

## Run it

```bash
npm install
npm run dev
```

Then open http://localhost:3000.
